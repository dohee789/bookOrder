# Book Order App
✅ React와 firebase database를 활용하여 책 정보를 불러오고 주문하는 앱<br>
✅ 원하는 책의 개수를 선택하고 add 버튼을 누르면 장바구니에 추가<br>
✅ 장바구니에 담아둔 책 리스트와 장바구니 합계 총합 출력<br>
